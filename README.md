# 世界征服者二编辑器(ing)

# 起因

- 想做一个简单的、可以在平板上直接调整好直接玩、不需要打字输入、界面美观、上手容易的编辑器；
- 想练习python技术

# 介绍

- 服务于Windows 平板、安卓平板（安卓可以装Qpython）
- 基于pyside2编写
- 全按钮（下拉弹窗）式操作
- 修改即玩

# 目的

- 还没想好、、、到哪算哪吧
- 希望和大佬们配合深入修改bin啥的
- 甚至想做一个综合的全面修改的大工具
- 欢迎大家提意见

- 修改最基础的战役配置方面、、、
